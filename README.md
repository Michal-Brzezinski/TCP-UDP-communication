# PROJEKT 1

## Podstawowe informacje do projektu:

W każdym katalogu (który jest modułem) wykonuję np.:
`go mod init 3.0-ipv4-tcp-server`
`go mod tidy`
Aby utworzyć pliki `.mod` (jednorazowo).

Aby uruchomić pliki klient-serwer oraz zadanie na 4.0  z raw socketem należy uruchomić komendę:
`go run main.go`

Aby uruchomić plik
z katalogu `4.5-capabilities` należy wykonać polecenie `./4.5-capabilities-raw-client`
Oraz serwer UDP z katalogu `3.0-ipv4` należy wykonać:
`./udp-server`
